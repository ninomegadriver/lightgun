AdvanceMAME Mouse Tester

1 Synopsis 

advm [-device_mouse DEVICE] [-log] [-logsync]

2 Description

The `advm' utility tests the mouse.

Check the `advdev.txt' file for the description of the `device_mouse' and related options.

In Linux you need to kill GPM before running `advm'. Otherwise the mouse may not respond correctly.
