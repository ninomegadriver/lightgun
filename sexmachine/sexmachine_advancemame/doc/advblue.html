AdvanceMAME Bluetooth Daemon

1 Synopsis 

advblue [-f]

2 Description

The `advblue' utility is a daemon that is able to pair and connect automatically bluetooth joysticks.

It uses the /usr/bin/bluetoothctl tool to communicate with the devices.

To start it, just run it without any argument.

3 Options

These are the command line options supported:
-f
Run in foreground mode. The interation with the bluetoothctl program is printed on the terminal.
